This project proposes a software-based AI-driven system to optimize urban traffic flow. Using real time traffic data, the system dynamically adjusts traffic light timings, predicts congestion patterns, and provides actionable insights for city planners. The solution integrates machine learning models with visualization tools, creating a scalable and fully software-driven solution that aligns with smart city initiatives.

Traffic congestion at signals is often exacerbated by static or manually controlled timers that allocate equal time to all lanes, regardless of actual traffic conditions. This inefficiency leads to unnecessary delays, excessive fuel consumption, and increased air pollution, negatively impacting both commuters and the environment. Our proposed solution introduces a smart, adaptive traffic signal system that dynamically adjusts signal durations based on real-time traffic conditions. Using vehicle detection and classification, the system estimates the number and types of vehicles in each lane and calculates the optimal crossing time required. This dynamically updated signal timing ensures smoother traffic flow, reducing wait times, fuel wastage, and emissions. The process continuously repeats for each lane, ensuring maximum efficiency and minimal congestion.
